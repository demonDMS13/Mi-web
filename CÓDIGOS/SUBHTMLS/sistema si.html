SISTEMA INTERNACIONAL
DE UNIDADES (SI)

Introducción

El Sistema Internacional de Unidades (SI) es el sistema de unidades más utilizado en el mundo. Se basa en siete unidades base y se utiliza en la mayoría de los campos científicos y técnicos.

Unidades Base

  • Metro (m) - Unidad de longitud
  • Kilogramo (kg) - Unidad de masa
  • Segundo (s) - Unidad de tiempo
  • Amperio (A) - Unidad de corriente eléctrica
  • Kelvin (K) - Unidad de temperatura termodinámica
  • Mole (mol) - Unidad de cantidad de sustancia
  • Candela (cd) - Unidad de intensidad luminosa
